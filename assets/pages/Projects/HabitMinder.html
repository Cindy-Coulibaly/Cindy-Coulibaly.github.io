Habit Minder

Habit Minder, is a group project and a work in progress for my application development II class. This app is an android application, which allows the user to create various habits that they want to do in a certain frequency. Their frequency will be counter using the streak method. Also this app allows the user to add notes but this feature is only accessible if the user in loged in. If not they will have to login or signUp.
